    >py -3 -m pip install requests  #安装requests模块

POST和GET请求方法  

    >GET- 从指定的资源请求数据；  
    >POST- 向指定的资源提交要被处理的数据；  

    其他HTTP请求方法：  
    >HEAD- 与GET相同，但只返回HTTP报头，不返回文档主体  
    >PUT- 上传指定的URI表示  
    >DELETE- 删除指定资源  
    >OPTIONS- 返回服务器支持的HTTP方法  

JSON是一种存储和交换文本信息的语法，类似XML。是目前最流行的数据交换方式。
